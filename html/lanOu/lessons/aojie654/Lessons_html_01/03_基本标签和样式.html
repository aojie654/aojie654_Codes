内部样式表，写在html中的head标签里;

行间样式

Eg.

              <pre background-color:yellow;"> </pre>
              写在标签内部的style属性，只对当前标签有效;

标签选择器: 类型名{样式}

Eg.

          
          pre{
            background-color: #gray;
          }

类型选择器:.类型名{样式}

Eg.

          .preClass0{
            background-color: orange;
          }
          

          相当于给标签起别名，通过别名选中标签设置样式(给标签起名:class="名称");
          命名规范:
            1.不能以数字开头(规则);
            2.起名要见名知意;
            3.驼峰命名法(多个单词中，从第二个单词起首字母大写,eg.:studentNumberOfMyClass);

特点：

          
          .preClass0{
            background-color: orange;
          }

          .preClass1{
            background-color: red;
          }
          

            1.不同标签可以拥有相同的类型名;
            2.统一标签可以拥有多个类型名(不同名之间用空格隔开);
            3.当多个类型选择器设置相同样式时，在样式表中最后位置上的样式生效(样式叠加效果：代码从上到下依次叠加执行);

id选择器: #id名称{样式}

Eg.

          
          #l9527{
            background-color: #999999;
          }
        

          id是一个标签的唯一身份标识不许重复，前端规范要求，不能用id选择器给盒子设置样式;
